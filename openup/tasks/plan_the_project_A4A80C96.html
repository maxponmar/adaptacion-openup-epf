Task: Planear el Proyecto
Implica la creación de un plan de proyecto detallado que establece los objetivos, alcance, recursos, plazos y entregables del proyecto de tesis, centrándose en el desarrollo de la plataforma.
Disciplines: Project Management
Purpose

El propósito de esta tarea es proporcionar una estructura organizativa y una guía para la ejecución exitosa del proyecto de tesis. El plan ayuda a definir claramente los pasos a seguir y asegura que los recursos se utilicen eficientemente para lograr los objetivos del proyecto.

        Main Description

        La planificación del proyecto es una etapa crítica que involucra una serie de actividades clave:

        • Definición de Objetivos y Alcance: Se establecen los objetivos específicos del proyecto de tesis, así como los límites del alcance. Esto incluye identificar las funcionalidades y características clave del sistema computacional a desarrollar.
        • Identificación de Recursos: Se determinan los recursos necesarios para el proyecto, incluyendo personal, hardware, software y cualquier otra infraestructura requerida. Esto asegura que los recursos estén disponibles cuando se necesiten.
        • Establecimiento de Plazos y Cronograma: Se crea un cronograma que detalla las actividades del proyecto y establece fechas límite para cada tarea. Esto proporciona una visión clara de la secuencia de trabajo y garantiza que el proyecto se complete a tiempo.
        • Asignación de Responsabilidades: Se asignan roles y responsabilidades a los miembros del equipo, incluyendo al Arquitecto, el autor de la tesis y cualquier otro participante. Cada persona sabe qué se espera de ella en el proyecto.
        • Gestión de Riesgos: Se identifican los posibles riesgos que podrían surgir durante el proyecto y se desarrollan estrategias para mitigarlos o abordarlos si se presentan.
        • Presupuesto: Se crea un presupuesto que detalla los costos estimados del proyecto, incluyendo gastos de desarrollo, recursos humanos y otros costos relacionados.
        • Comunicación y Colaboración: Se establece un plan de comunicación que describe cómo se compartirá la información, se reportarán los avances y se gestionarán las interacciones dentro del equipo de proyecto.
        Steps
        Definir Objetivos y Alcance del Proyecto
        En este paso, se establecen los objetivos específicos del proyecto de tesis, así como los límites del alcance. Se define claramente lo que se espera lograr y lo que está fuera del alcance del proyecto.
        Identificar Recursos Necesarios

        Se determinan los recursos requeridos para el proyecto, que pueden incluir personal, hardware, software, acceso a bases de datos, herramientas específicas, entre otros. Esto asegura que los recursos estén disponibles cuando se necesiten.

        Establecer Plazos y Cronograma

        En este paso, se crea un cronograma detallado que establece las fechas de inicio y finalización de cada tarea y actividad del proyecto. Esto proporciona una guía temporal para la ejecución del proyecto.

        Asignar Responsabilidades

        Se asignan roles y responsabilidades a los miembros del equipo de proyecto, incluyendo al Arquitecto, el autor de la tesis y otros participantes. Cada persona sabe claramente qué se espera de ella en el proyecto.

        Gestionar Riesgos

        Se identifican los posibles riesgos que podrían surgir durante el proyecto, como retrasos, cambios en los requisitos o problemas técnicos. Se desarrollan estrategias para mitigarlos o abordarlos si ocurren.

        Definir Plan de Comunicación y Colaboración

        Se establece un plan de comunicación que describe cómo se compartirá la información, se reportarán los avances y se gestionarán las interacciones dentro del equipo de proyecto. La colaboración efectiva es clave para el éxito.

        Key Considerations

        Gain agreement with stakeholders and the rest of the project team regarding the order of objectives and the duration of the project and make adjustments as necessary.
