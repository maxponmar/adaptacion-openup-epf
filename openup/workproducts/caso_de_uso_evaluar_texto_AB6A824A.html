Artifact: Caso de Uso: Evaluar Texto
Permite al usuario evaluar una sección específica de su tesis, como justificación, conclusiones, objetivos, etc., y recibir una puntuación basada en el contenido de esa sección.
Purpose

Evaluar texto de tesis.

Description
Main Description

Actores: Usuario, Sistema de Evaluación.

Flujo Principal:

  1. El Usuario inicia sesión en la aplicación.
  2. El Usuario navega a la sección "Evaluación de Tesis".
  3. El Usuario selecciona la sección de su tesis que desea evaluar (por ejemplo, "Justificación").
  4. El Sistema de Evaluación analiza el contenido de la sección seleccionada y genera una puntuación.
  5. El Usuario revisa la puntuación y la retroalimentación proporcionada por el Sistema de Evaluación.
  6. El Usuario decide si realizar cambios en la sección evaluada para mejorar la puntuación.
  7. Si el Usuario decide hacer cambios, edita la sección correspondiente.
  8. El Usuario guarda los cambios realizados.
  9. El Usuario puede volver a evaluar la sección después de realizar las modificaciones.

Escenarios Alternativos:

Si el Usuario no está satisfecho con la puntuación o la retroalimentación, puede solicitar más detalles o aclaraciones al Sistema de Evaluación.
Si el Usuario decide no realizar cambios, la puntuación original se conserva.


Precondiciones: El Usuario debe haber iniciado sesión en la aplicación y tener una tesis cargada con al menos una sección.

Postcondiciones: Se registra la puntuación y, si corresponde, los cambios realizados en la sección evaluada.
